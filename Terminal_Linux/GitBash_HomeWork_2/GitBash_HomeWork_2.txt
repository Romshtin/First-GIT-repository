GIT Homework 1

Для выполнения задания у вас должен быть установлен для Windows - GitBash.
Создан аккаунт в GitHub
Создан репозиторий в GitHub
Созданный репозиторий склонирован на локальный компьютер в отдельную папку.

Все шаги сценария выполняйте в терминале GitBush, Terminal, в папке под гитом.


Как отправить ДЗ на проверку.
 1. Создайте текстовый файл как в первом ДЗ по Terminal.
 2. Сценарий перенесите в этот файл.
 3. Напротив каждого действия - напишите команду в GitBash

Файл со сценарием и ссылку на свой гит хаб отправляйте менторам на проверку.

 4. Создать файл “new.json” -> "> new.json"
 5. Добавить файл под гит. -> "git add new.json"
 6. Закоммитить файл. -> "git commit -m 'created new.json'"
 7. Отправить файл на внешний GitHub репозиторий. -> "git push"
 8. Создать файлы new2.json, new2.xml, new2.txt -> "touch new2.json new2.xml new2.txt"
 9. Добавить файлы под гит. -> "git add ."
 10. Закоммитить файлы. -> "git commit -m 'created files"
 11. Отправить файлы на внешний GitHub репозиторий. -> "git push"
 12. Отредактировать содержание файла “new2.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата).

 "'vim new2.txt' + 'i' + 'Kirillov Roman Aleksandrovich, age 25, 1 pet, 60000 rubles' + 'Esc' + ':wq'"

 13. Отредактировать содержание файла “new2.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON. 

 "'subl new2.json' + 
 '{
        "name": "Kirillov Roman Aleksandrovich",
        "age": 25,
        "pets": 1,
        "desired salary": "60000 rubles"
 }' 
 + 'File' + 'Save' + 'Закрыть'"

 14. Добавить и закоммитить “new2.txt” одной строчкой. -> "git add new2.txt && git commit -m 'changes'"
 15. Добавить и закоммитить “new2.json” одной строчкой. -> "git add new2.json && git commit -m 'changes'"
 16. Отправить изменения на внешний репозиторий. -> "git push"
 17. Отредактировать содержание файла “new2.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML.

 "'subl new2.xml' +
 '<?xml version="1.0" encoding="UTF-8"?>
	<root>
		<Name>Kirillov Roman Aleksandrovich</Name>
		<Age>25</Age>
		<Pets>1</Pets>
		<Desired_salary>60000 rubles</Desired_salary>
	</root>'
 + 'File' + 'Save' + 'Закрыть'"

 18. Добавить и закоммитить “new2.xml” одной строчкой. -> "git add new2.xml && git commit -m 'changes in new2.xml'"
 19. Отправить изменения на внешний репозиторий. -> "git push"
 20. Отправить предыдущее домашнее задание по GitBash Terminal на внешний репозиторий.

 "'mv ~/Desktop/GitBash_HomeWork_1.txt ./' + 'git add Git_HomeWork_1.txt && git commit -m 'adding homework' && git push'"

 21. В веб интерфейсе GitHub создать файл new3.txt -> "'Add file' + 'Create new file' + 'new3.txt' + 'Commit new file'"
 22. Отредактировать в веб интерфейсе содержание файла “new3.txt” - написать информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, страна которую хотели бы посетить).

 "'new3.txt' + 'Edit this file' + 'Мост в Терабитию, Декстер, Пангасиус обжаренный в муке и яйце, лето, Канада.'"

 23. Сделать Commit changes (сохранить) изменения на веб интерфейсе. -> "'Проврутить страницу вниз' + 'Commit changes'"
 24. Синхронизировать внешний и локальный репозитории (слить изменения с внешнего репозитория) -> "git pull"
 25. Отредактировать в веб интерфейсе содержание файла “new2.json” - добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, страна которую хотели бы посетить). Всё в формате JSON.
 "'new2.json' + 'Edit this file' + 
 '				,
"favourite film": "Мост в Терабитию",
	"favourite serial": "Декстер",
	"favourite food": "Пангасиус обжаренный в муке и яйце",
	"favourite time of year": "лето",
	"country to visit": "Канада"'"
 

 26. Сделать Commit changes (сохранить) изменения на веб интерфейсе. -> "'Проврутить страницу вниз' + 'Commit changes'"
 27. Синхронизировать внешний и локальный репозитории -> "git pull" 
 28. Отредактировать в веб интерфейсе содержание файла “new2.xml” - добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, страна которую хотели бы посетить). Всё в формате XML.
 "'new2.xml' + 'Edit this file' + 
 '
 		<Favourite film>Мост в Терабитию</Favourite film>
		<Favourite serial>Декстер<Favourite serial>
		<Favourite food>Пангасиус обжаренный в муке и яйце</Favourite food>
		<Favourite time of year>лето</Favourite time of year>
		<Country to visit>Канада</Country to visit>'"

 29. Сделать Commit changes (сохранить) изменения на веб интерфейсе. -> "'Проврутить страницу вниз' + 'Commit changes'"
 30. Синхронизировать внешний и локальный репозитории -> "git pull" 

Данные для заполнения и редактирования файлов можете использовать любые, не обязательно свои.
